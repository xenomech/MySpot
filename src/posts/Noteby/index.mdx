## 1. Setting up the dev environment

This is a fairly basic topic, we need to setup a bunch of tools before starting the project.

1. nodejs and npm

   For ubuntu/debian :

   ```sh
   sudo apt-get install nodejs npm
   ```

   For Manjaro/Arch :

   ```sh
   sudo pacman -S nodejs npm
   ```

   For windows :

   `https://nodejs.org/en/`

2. gatsby-cli

   ```sh
   npm install -g gatsby-cli
   ```

3. git

   Install git for version control

4. VScode

   Download and install the latest version of VScode from

   `https://code.visualstudio.com/#alt-downloads`
